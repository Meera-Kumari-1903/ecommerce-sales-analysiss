# E-Commerce Sales Analysis

## Project Overview

This project analyzes customer purchasing behavior, sales performance, payment methods, and product categories using SQL and Python.

The analysis is performed on the Olist Brazilian E-Commerce Dataset containing over 100,000 orders.

## Objectives

- Analyze sales performance
- Identify top-performing products
- Study customer behavior
- Understand payment preferences
- Generate business insights

## Dataset

Source:
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Tools & Technologies

- Python
- SQL (DuckDB)
- Pandas
- Matplotlib
- Google Colab
- GitHub

## Business Questions

1. What is the total revenue generated?
2. Which cities have the highest number of customers?
3. What are the most popular payment methods?
4. Which product categories generate the highest sales?
5. What is the monthly sales trend?
6. Which states contribute the most revenue?

## Key Insights

- Most orders were successfully delivered.
- Credit cards were the most preferred payment method.
- A few cities contributed a large portion of total sales.
- Some product categories significantly outperformed others.
